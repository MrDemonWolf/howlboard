## What is HowlBoard?

HowlBoard is a **self-hosted collaborative whiteboard** built on [Excalidraw](https://excalidraw.com) and deployed on Cloudflare's edge infrastructure (Workers, D1, R2, KV). It gives you full ownership of your data while providing a rich drawing and collaboration experience.

## Why HowlBoard?

- **Self-hosted** -- Your data stays on your infrastructure. No third-party SaaS required.
- **Edge-native** -- Deployed on Cloudflare Workers for low-latency access worldwide.
- **Built on Excalidraw** -- The same powerful drawing engine used by millions, with persistent storage on top.
- **Simple deployment** -- A single Cloudflare Worker handles the API; D1 stores metadata, R2 stores drawings.

## Features

- **Persistent Storage** -- Drawings saved to Cloudflare R2 (up to 10 MB per scene)
- **Real-time Collaboration** -- Multiple users can draw simultaneously
- **Share Links** -- Generate links with view or edit permissions, optional expiration, and max-use limits
- **Collections** -- Organize boards into folders
- **Admin Dashboard** -- Manage users, boards, and system settings
- **Two-Factor Authentication** -- Optional TOTP-based 2FA for user accounts
- **Role-based Access** -- Admin and viewer roles; first registered user becomes admin
